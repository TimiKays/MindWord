## 如何参与 / How to Contribute

### 反馈问题 / Report Issues
- 遇到 bug 直接在 GitHub Issues 提
- 描述清楚复现步骤就好
- 看到错别字或界面问题也可以提

- Report bugs directly on GitHub Issues
- Describe the reproduction steps clearly
- Typos or UI issues are also welcome

### 功能建议 / Feature Suggestions
- 有什么想法直接在 Issues 说
- 如果能简单描述使用场景更好
- UI/UX 建议也很欢迎

- Share your ideas directly in Issues
- Better if you can describe the use case
- UI/UX suggestions are welcome too

### 代码贡献 / Code Contributions
- 欢迎提交 PR，但我可能会比较慢才处理（个人项目，时间有限）
- 建议先在小问题上试水
- 代码风格保持和现有的一致就行

- PRs are welcome, but I might be slow to process them (personal project, limited time)
- Start with small issues first
- Keep code style consistent with existing code

### 测试帮忙 / Testing Help
- 在不同浏览器试试功能
- 手机端也帮忙看看显示问题
- 发现什么奇怪的地方就告诉我

- Test features in different browsers
- Help check display issues on mobile
- Let me know if you find anything weird
