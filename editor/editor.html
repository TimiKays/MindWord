自动保存 - Ctrl+S 手动保存
📚 帮助文档

⌨️ 快捷操作

  • 点击节点：选中（高亮）
  • 双击节点：编辑
  • Tab：添加子节点
  • Enter：添加同级节点
  • 拖拽节点 / 框选 / Space+拖动画布
  • Ctrl + 滚轮：放大/缩小画布
  • Delete/Backspace：删除选中节点
  • Ctrl + S：保存到本地存储
  • Ctrl + V：粘贴图片

📝 支持的Markdown格式

节点类型（会生成思维导图节点）：

  • # 标题 - 一级标题
  • ## 标题 - 二级标题
  • ### 标题 - 三级标题
  • - 无序列表 - 列表项
  • 1. 有序列表 - 编号列表

非节点格式（不影响思维导图）：

  • 普通段落文本
  • **粗体** - 粗体
  • *斜体* - 斜体
  • `代码` - 行内代码
  • > 引用 - 引用块
  • ![图片](url) - 图片
  • [链接](url) - 超链接
  • 表格、代码块等

🖼️ 图片支持

  • 直接粘贴图片（Ctrl+V）
  • 拖拽图片文件到编辑器
  • 使用 Markdown 语法插入
  • 导出时自动打包所有图片

💾 保存功能

  • 实时自动保存到浏览器存储
  • Ctrl+S 手动保存
  • 重新打开页面自动恢复内容
  • 支持导出 .md 文件或 .zip 包

🔧 调试功能

  • 查看JSON：查看解析后的数据结构
  • 支持上传 .md 或 .zip 文件
  • 自动处理图片引用关系
