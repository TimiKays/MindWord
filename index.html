核心功能

MindWord 重新定义了结构化写作体验。通过将 Markdown结构与思维导图节点实时映射，实现可视化创作流程，让创作思路与正文完美融合。 配合 AI辅助功能，轻松完成头脑风暴、主题拆解、段落优化，最终生成文档。

Markdown编辑器

强大的数据收集与初始化功能，支持多种导入方式，让你从任何起点开始创作

支持多种数据来源

  • • 粘贴markdown纯文本
  • • 粘贴富文本并自动转为Markdown
  • • 粘贴图片
  • • 上传md/zip（带图片）

导入/导出源文件

支持导入/导出标准语法的md/zip（带图片），可反复修改与复用，不用担心被工具捆绑。

数据本地离线存储

所有数据本地存储（无服务器），安全可控，保护你的隐私和知识产权。

思维导图画布

可视化同步编辑文档结构，让你的创意和思路更加清晰有序

树形节点定义

Markdown 所有1到6级标题与有序/无序列表行作为节点标题，其他文本均作为节点的备注属性。

双向同步编辑

思维导图节点与Markdown文档标题双向同步，任意一端修改，另一端实时同步。

AI 生成子节点

在头脑风暴或细化主题时，可让AI基于某个节点一键生成子节点，并可手动采纳或编辑。

Word文件生成

输出日常办公文档形成闭环，满足正式场合的文档需求

预设模板

不需额外配置，直接基于预设模板样式一键生成word文件，快速满足基本需求。

自定义模板

对word样式有要求的用户，可自行下载模板并修改样式集，上传模板生成符合特定格式的文档。

完整保留结构

生成的Word文档完整保留思维导图中的层级结构，格式规范，美观专业。

快速上手

只需简单几步，即可开始使用MindWord提升你的写作效率

1

初始化

上传 Markdown文件，或复制粘贴markdown或富文本，即可看到对应预览与思维导图。

示例 Markdown 片段

导入下面的 Markdown 片段会形成对应树状结构：

# 产品需求文档
## 功能概述
- 背景说明
- 目标用户
## 功能详情
### 登录
- 支持手机号
- 支持第三方
### 写作编辑
- 思维导图视图
- Markdown 编辑器
2

编辑方式

在右侧思维导图中双击节点编辑标题或备注，拖拽调整结构；在左侧 Markdown 编辑器中编辑对应部分，任一侧修改都会同步到另一侧。

常用快捷与操作

• 新建子级节点：选中节点后按Tab
• 新增同级节点：选中节点后按 Enter
• 删除节点：选中后节点按 Delete
• 撤销：Ctrl+z，可撤销最近10次
• 重做：Ctrl+Shift+Z
• 多选：支持鼠标框选
3

导出

Markdown可导出为md文件，如果有图片将一起打包成zip文件。也可以直接导出为格式化的Word文档。

隐私保障

我们重视你的数据安全和隐私保护，让你使用得更加放心

数据存储

未登录用户：所有数据完全本地存储，不上传到任何服务器。
登录用户：可选择本地存储或使用云同步功能。

AI功能说明

使用AI生成功能时，必要的上下文信息（如当前节点文本）会传输给对应的AI平台处理，但不作存储。

隐私承诺

除云同步功能外，其他所有功能均可免登录使用和完全本地存储。实现最小化数据收集，最大化用户隐私保护。 隐私政策>>
